# PINN IVD Poromechanics

Physics-Informed Neural Network (PINN) for Intervertebral Disc poromechanics modeling based on Triphasic Theory.

## Features

- Triphasic Theory implementation
- NP/AF region with anisotropic fiber modeling
- Porosity-based permeability with safety clamp
- Adaptive residual sampling (RAS)
- COMSOL-aligned boundary conditions

## Requirements

```
Python 3.8+
PyTorch
NumPy
Matplotlib
```

## Usage

```bash
python main.py
```
